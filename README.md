# SIEM — Системы управления информацией и событиями безопасности

> Компетенции в области SIEM, полученные в рамках курса «Программно-аппаратные средства защиты информации» в РГУ нефти и газа им. И.М. Губкина.

**Репозиторий:** https://github.com/goloshumovmisha2005-design/siem-labs

---

## Компетенции

### Работа с инцидентами безопасности
- Просмотр и анализ характеристик инцидентов: время, тип, источник, уровень критичности.
- Фильтрация, сортировка и группировка событий.
- Анализ цепочек событий для выявления аномалий и атак.
- Понимание структуры инцидента: корреляционные правила, исходные события, метаданные.

### Создание и настройка отчётов
- Настройка параметров отчётов (период, типы событий, фильтры).
- Экспорт результатов в различных форматах (PDF, CSV, HTML).
- Визуализация статистики инцидентов (графики, диаграммы).
- Настройка периодических отчётов для мониторинга безопасности.

### Администрирование SIEM
- Управление источниками событий (Windows, Linux, сетевое оборудование).
- Настройка правил корреляции для автоматического обнаружения угроз.
- Работа с компонентом Manager and Configuration.
- Понимание принципов корреляции событий и выявления аномалий.
